# DooTask 帮助文档

这是一个基于 VitePress 的文档生成项目。

可以修改`/docs/.vitepress/config.ts` 文件来自定义文档配置。

## 安装与使用

1. 克隆项目：

```bash
git clone <repository-url>
```

2. 安装依赖：

```bash
npm install
```

3. 运行文档开发服务器：

```bash
npm run docs:dev
```

4. 构建生产环境文档：

```bash
npm run docs:build
```

5. 预览已构建的文档：

```bash
npm run docs:preview
```
